## WAI-ARIA Roles, States, and Properties

### Tab

✅ 탭 버튼 그룹은 `role="tablist"`로 설정되어 있습니다.

### Tab Item

✅ 탭 버튼은 `role="tab"`으로 설정되어 있습니다.  
✅ 선택된 탭 버튼에는 `aria-selected="true"`가 설정되어 있습니다.  
✅ 탭 버튼에는 제어 대상 패널의 ID를 참조하는 `aria-controls`가 설정되어 있습니다.

### Tab Panel

✅ role="tabpanel"으로 설정되어 있습니다.  
✅ 패널 요소는 고유한 ID를 가지며, 탭 버튼의 `aria-controls`가 이를 참조합니다.  
✅ `aria-labelledby`로 연결된 탭 버튼의 ID를 참조합니다.

## Usage

```tsx
import { useState } from 'react';

import { Tabs } from '@useplease-ui/ppudding';

export default function TabExample() {
  const [value, setValue] = useState(1);

  const handleChange = (newValue: number) => {
    setValue(newValue);
  };

  return (
    <Tabs value={value} onChange={handleChange}>
      <Tabs.Tab aria-label="navigation">
        <Tabs.Item index={1} label="1" />
        <Tabs.Item index={2} label="2" />
        <Tabs.Item index={3} label="3" />
      </Tabs.Tab>
      <Tabs.Panel index={1}>
        <p>Tab Panel #1</p>
      </Tabs.Panel>
      <Tabs.Panel index={2}>
        <p>Tab Panel #2</p>
      </Tabs.Panel>
      <Tabs.Panel index={3}>
        <p>Tab Panel #3</p>
      </Tabs.Panel>
    </Tabs>
  );
}
```
